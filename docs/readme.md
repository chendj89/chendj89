### 引用文件

- 引用整个文件

```md
@[code](@docs/scss/mixin.scss)
```

- 引用文件中的单个或多个方法

```md
@[code {btn}](@docs/scss/mixin.scss)
```

```md
@[code {btn,input}](@docs/scss/mixin.scss)
```

```md
@[code {union}](@docs/.vuepress/tool/array/union.js)
```
