## Vue2.7 升级指南

### <vp-word name="交集">intersection</vp-word>

- 方法封装

```js export
export const intersection = (a, b, key = null) => {
  return a.filter((i) =>
    key ? b.map((i) => i[key]).includes(i[key]) : b.includes(i)
  );
};
```
```js export
export const getName=(name)=>{
  return name+"🐧";
}
```

const a = [
  { id: 1, name: "张三" },
